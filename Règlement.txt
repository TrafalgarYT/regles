1) Pas de Freekill (Tuer une personne sans raison RP valable) = ban permanent.
2) Pas de Prop-block, il faut que des individus puisse avoir un moyen d'accéder de l'autre côté du prop pour cela veuillez placer un bouton / keypad des deux côtés.
3) Pas de Spam (Micro/prop)
4) Pas d'insulte /OOC, vous devez respecter les autres joueurs. 
Les insultes peuvent être introduite dans un roleplay mais elles doivent être limité, veuillez ne pas insulter les mamans toutes les secondes. 
5) Pas de Metagaming. Si nous avons enlever les noms et métier au dessus de la tête des gens et dans le menu tab, c'est pas pour que vous vous amusiez a faire n'importe quoi.
6) Pas de HRP vocal. Faites /ooc si vous souhaitez posez une question hrp a la communauté ou sinon contacter un admin via /// "message"
7) Interdiction d'exploiter des bugs, exemple : s'asseoir sur un props afin de passer a travers un mur.
8) Pas de Props Kill = ban permanent.

			Si vous souhaitez intégrez les forces de police, il vous faudras vous renseigner auprès d'un admin/policier pour qu'il vous donne les infos nécessaires.

Votre personnage doit ressentir les mêmes émotions que vous pouvez ressentir IRL, si vous vous faites braquer par un gars armé, vous devez être soumis a ce qu'il vous dit.
Essayez de bien respecter le règlement sinon en cas de problème, vous pourrez prendre des avertissement puis sanction.
